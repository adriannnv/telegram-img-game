# natsort - Notes

## What is it?
`natsort` is a python library that implements natural sorting as in sorting that makes sense to the human eye and not a computer's usual way of doing it... for example sorting `[1, 10, 2]` with `natsort` would yield the expected result and with normal sorting it wouldnt change the array.

ns.PATH is natsort but for paths, splitting paths like messages.html to messages . html, comparing them properly( eg: messages2.html and messages.html)
